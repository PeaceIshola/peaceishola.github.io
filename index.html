Research Interest

My research investigates context-aware networking and adaptive communication protocols in real-time collaborative 3D environments. The primary research interest centers on dynamic protocol selection based on interaction context, where the system intelligently chooses between WebSocket and WebRTC protocols depending on data type (video, control, drawing, sensor), urgency levels (critical vs routine), network conditions (latency, packet loss, bandwidth), and interaction semantics (Rubik's cube manipulation, whiteboard drawing, chat messaging). The platform implements four distinct operational modes: hybrid (combining protocols), pure WebRTC (peer-to-peer), pure WebSocket (client-server), and the most innovative context-aware mode that dynamically routes communications based on real-time performance metrics and user interaction patterns. The research methodology employs a comprehensive experimental design that enables controlled protocol switching, network condition simulation, and real-time metrics collection through Supabase integration for performance validation. The platform serves as a testbed for studying multi-user 3D interactions, spatial awareness in virtual environments, and real-time state synchronization across distributed users. Key innovations include adaptive communication architecture that considers interaction semantics and network performance simultaneously, multi-modal communication with seamless protocol switching based on content type and quality requirements, and real-time analytics for predictive routing and user experience optimization. The research applications span collaborative virtual reality, educational technology, gaming and entertainment, and industrial applications such as remote collaboration in 3D design and virtual training environments, ultimately advancing the state-of-the-art in distributed collaborative systems and adaptive communication architectures.
